# MobileApp-SWProject
This is the mobile app for the software project for the 2026 software fair of the UTFSM, Concepción Site, of Vicente Gutiérrez Mardones, Rodrigo López Ponce and Felipe Riveros Muñoz.

After you clone this repository, execute **npm install** on your terminal, open the folder where you want to run the app (**app** or **test**), execute **npx expo start** on your terminal and y'all be ready :).
